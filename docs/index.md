# VaultKeeper

VaultKeeper is a local Android password manager created as a PMVS team project. The app demonstrates secure local credential storage, encryption, and quality engineering practices.

## Project task summary
The project goal is to build a mobile application that securely stores credentials in an offline-first vault, protects access with a master password, and supports password breach checks through an external API.
